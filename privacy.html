Политика конфиденциальности

Приложение «Псалтирь» (Orthodox Psalter)

Дата вступления в силу: 11 апреля 2026 г.

1. Сбор данных

Приложение «Псалтирь» не собирает, не хранит и не передаёт никаких персональных данных пользователей. Учётные записи не требуются.

2. Хранение данных

Все пользовательские данные (закладки, заметки, настройки, имена для поминовений, подборки псалмов, прогресс правила чтения) хранятся локально на устройстве пользователя и не передаются на внешние серверы.

3. Сетевые запросы

Тексты переводов, толкований, молитв и интерфейс приложения работают полностью офлайн и встроены в приложение.

Сетевое подключение используется только по инициативе пользователя для скачивания аудио-озвучивания псалмов и молитв (опциональная функция). Аудио-файлы загружаются с GitHub Releases и зеркала Cloudflare R2. При этих запросах сервер видит только стандартные технические данные (IP-адрес, User-Agent), которые не сохраняются и не связываются с вашей личностью.

После скачивания аудио хранится локально и воспроизводится без подключения к интернету.

4. Аналитика и реклама

Приложение не содержит рекламы, трекеров, аналитических SDK, crash-репортеров и иных средств отслеживания.

5. Разрешения

Приложение запрашивает только следующие разрешения:

  • Доступ к интернету — для опционального скачивания аудио-файлов;
  • Foreground service — для работы аудио-плеера в фоне;
  • POST_NOTIFICATIONS (Android 13+) — для отображения уведомлений плеера и прогресса загрузки.

6. Сторонние сервисы

При скачивании аудио используются:

  • GitHub (политика конфиденциальности) — для раздачи аудио-архивов;
  • Cloudflare R2 (политика конфиденциальности) — для зеркала.

7. Дети

Приложение не собирает данных и безопасно для пользователей любого возраста.

8. Изменения политики

Изменения публикуются по этому адресу с обновлением даты вступления в силу.

9. Контакт

По вопросам конфиденциальности: GitHub Issues

Privacy Policy

Application "Псалтирь" (Orthodox Psalter)

Effective date: April 11, 2026

1. Data Collection

Orthodox Psalter does not collect, store, or transmit any personal data. No accounts are required.

2. Data Storage

All user data (bookmarks, notes, settings, commemoration names, psalm collections, reading schedule progress) is stored locally on the user's device and is never sent to any external servers.

3. Network Requests

Texts of translations, commentaries, prayers, and the application interface work fully offline and are bundled within the application.

Network access is used only at user's explicit request to download optional audio narration of psalms and prayers. Audio files are downloaded from GitHub Releases and a Cloudflare R2 mirror. These requests expose only standard technical data (IP address, User-Agent) which is not stored or linked to your identity.

Once downloaded, audio is stored locally and played back without an internet connection.

4. Analytics and Advertising

The application does not contain any advertising, trackers, analytics SDKs, crash reporters, or any other tracking mechanisms.

5. Permissions

The application requests only the following permissions:

  • Internet access — for optional downloading of audio files;
  • Foreground service — to keep the audio player running in the background;
  • POST_NOTIFICATIONS (Android 13+) — to display player and download progress notifications.

6. Third-Party Services

Audio downloads use:

  • GitHub (privacy policy) — for distributing audio archives;
  • Cloudflare R2 (privacy policy) — for the mirror.

7. Children

The application does not collect any data and is safe for users of all ages.

Updates will be posted at this URL with a new effective date.
